# Тестовое задание для Core Infrastructure

Клиент-сервер для передачи файлов по сети. Код сервера - в папке `server`, код клиента - в папке `client`.

Для сервера макрос `SAVE_DIR` определяет папку для сохранения файлов, `SERVER_PORT` - номер порта сервера, `SERVER_ADDR` - адрес сервера.

Для клиента макрос `FILE_PATH` определяет путь до отправляемого файла, `SAVED_FILE_NAME` - имя файла при сохранении на сервере,
`SERVER_PORT` - номер порта сервера, `SERVER_ADDR` - адрес сервера.
